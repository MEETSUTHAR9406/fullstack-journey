# Fullstack Journey

A simple practice repository for learning fullstack web development step by step.

This repository contains HTML, CSS, and JavaScript exercises, along with small frontend projects and UI clone practice.

## About

This repository is used to track my learning progress while building small examples, layouts, and interactive web pages.

Current focus:

- HTML structure and semantic elements
- CSS styling, layouts, animations, and responsiveness
- JavaScript basics, logic, functions, loops, arrays, and DOM practice
- Small frontend projects and UI clones

## Folder Structure

```text
fullstack-journey/
├── HTML/               # HTML practice files and topic-based examples
├── CSS/                # CSS files for styling practice
├── JS/                 # JavaScript practice scripts
├── media/              # Images, icons, audio, and video assets
├── netflix-ui-clone/   # Netflix UI clone practice
├── ultraedit-clone/    # UltraEdit website clone practice

```

## Projects

### To-Do App

A basic to-do application built with HTML, CSS, and JavaScript.

### Netflix UI Clone

A frontend UI clone inspired by Netflix, focused on layout and styling practice.

### UltraEdit Clone

A website clone practice project focused on recreating the UltraEdit interface.

### Spotify Intro Page

A static frontend practice page inspired by Spotify-style UI.

## Tech Used

- HTML
- CSS
- JavaScript
- Node.js package setup for JavaScript practice

## How to Use

Open any `.html` file directly in your browser to view the page.

For JavaScript files that use Node.js, run:

```bash
node filename.js
```

Install dependencies if needed:

```bash
npm install
```

## Purpose

The goal of this repository is to learn by building, practicing, and improving frontend development skills before moving deeper into fullstack development.
